DECISIONS MADE:
- Image pipeline: upload goes straight to Cloudinary for resize/compression
  and hosting/CDN; the resulting Cloudinary URL is stored in the database.
  Decided against also copying the file into Supabase Storage afterward --
  that would mean paying for storage and bandwidth twice for no benefit.
- Skills field is a simple free-text tag list (text[]), capped at 15 tags,
  no predefined taxonomy.
- Contact info uses a fixed set of optional fields: email, Instagram,
  Facebook, X, website -- not an open-ended list.
- Report reasons restricted to exactly: stolen_art, inappropriate, other.
